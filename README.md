# Pallet-Forge

1. No More Hex Code Hunting: Just drop an image in, and it instantly reads the pixels and finds the best, most dominant colors for you. It saves so much time and mental energy because you never have to manually look for hex codes again.

2. Instant Developer Handoff: This makes life incredibly easy if you code. Instead of manually typing out variables, you just click one button to download a JSON file. You can drop that file directly into another program or project and use the colors immediately.

3. Skips the Accessibility Headache: Instead of copying your colors into a completely different website to check if they are readable, it automatically tests every color against the others to tell you what's readable right there on the same screen.
